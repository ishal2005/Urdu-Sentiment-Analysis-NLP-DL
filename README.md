#  Urdu Sentiment Analysis using NLP & Deep Learning

A bilingual sentiment analysis system that classifies **Urdu** and **Roman Urdu** text into positive and negative sentiments using Machine Learning, Deep Learning, and Transformer-inspired models.

---

## Project Overview

This project was developed as part of my Artificial Intelligence coursework at COMSATS University Islamabad, Attock Campus.

The system supports:

- Native Urdu Script
- Roman Urdu
- Machine Learning Models
- Deep Learning Models
- Transformer-based Classification
- Interactive Gradio Web Interface

---

##  Features

- Dual-script sentiment analysis
- Automatic text preprocessing
- Urdu & Roman Urdu support
- Multiple AI model selection
- Confidence score prediction
- Interactive Gradio interface
- Performance visualization

---

## Models Used

### Machine Learning

- Logistic Regression
- Support Vector Machine (SVM)

### Deep Learning

- 1D CNN
- LSTM

### Transformer-based Models

- mBERT
- XLM-RoBERTa

---

##  Dataset

The system is based on the **Urdu Corpus for Sentiment Analysis (UCSA)**.

Dataset Statistics

- Total Reviews: 9,601
- Positive Reviews: 4,843
- Negative Reviews: 4,758
